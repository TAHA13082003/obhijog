# Obhijog 🎓
### A Web-Based Smart Campus Grievance Management System

## 🔗 Live Demo
👉 [smartgrieve.vercel.app](https://smartgrieve.vercel.app)

## About
Obhijog bridges the communication gap between students and 
university administration at Daffodil International University.

## ✨ Features
- ✅ Anonymous or named grievance submission
- ✅ Unique Tracking ID for every complaint
- ✅ Auto-routing to relevant departments
- ✅ Real-time status updates
- ✅ Admin analytics dashboard

## 🛠 Tech Stack
- **Frontend:** React.js + Tailwind CSS
- **Backend:** Django REST Framework
- **Database:** PostgreSQL
- **Auth:** JWT Token
